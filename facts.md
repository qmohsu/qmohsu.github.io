## Open-source contributions

### UrbanNav Dataset
- **What**: Open multisensory dataset for benchmarking urban navigation algorithms
- **Sensors**: GNSS, IMU, LiDAR, camera
- **Repository**: [github.com/IPNL-POLYU/UrbanNavDataset](https://github.com/IPNL-POLYU/UrbanNavDataset)
- **Publication**: "Hong Kong UrbanNav: An open-source multisensory dataset for benchmarking urban navigation algorithms", NAVIGATION (2023)

### GraphGNSSLib
- **What**: Factor-graph-based GNSS positioning / RTK research code (C/C++)
- **Repository**: [github.com/weisongwen/GraphGNSSLib](https://github.com/weisongwen/GraphGNSSLib)
- **Publication**: "Towards Robust GNSS Positioning and Real-time Kinematic Using Factor Graph Optimization", IEEE ICRA (2021)

### GPS_VT_SDR
- **What**: Open-source Matlab code for GPS vector tracking (software-defined receiver)
- **Source**: [NOAA GPS Toolbox](https://www.ngs.noaa.gov/gps-toolbox/GPS_VT_SDR.htm)
