GraphGNSSLib is an open-source C/C++ library that implements GNSS positioning using factor graph optimization. It provides an alternative to traditional Extended Kalman Filter (EKF) pipelines for GNSS and GNSS/INS integration.

The library supports pseudorange and carrier-phase measurements and is designed as a research platform for exploring constraint-aware positioning in urban environments.
