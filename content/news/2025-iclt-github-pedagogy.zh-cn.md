许立达教授于 2025 年 5 月 23 日在香港举行的**2025 年国际学与教会议（ICLT）**上宣读论文。该论文——*GitHub 作为辅助教学工具*——延续了本实验室多年来在工程教育中运用 GitHub 工作流进行学生项目管理、代码审查和版本控制的教学创新研究系列。

这一教学研究系列可追溯至许教授获得的香港理工大学在线教学研究资助（关于基于 GitHub 的教学法，96 万港元，2021-22 年）、URIS 本科教学资助（用于无人机仿真平台开发，10 万港元，2021-23 年），并最终促成 2024 年的[香港理工大学校长卓越教学奖](https://www.polyu.edu.hk/)（2024-26 年）。
