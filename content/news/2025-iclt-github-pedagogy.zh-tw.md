許立達教授於 2025 年 5 月 23 日在香港舉行的**國際學與教會議（ICLT）2025** 上發表論文。該論文——*GitHub 作為輔助教學工具*——延續了實驗室多年來的教學創新研究主線，運用 GitHub 工作流程於工程教育中的學生專案管理、程式碼審查及版本控制。

這條教學研究主線可追溯至許教授於 GitHub 教學法的香港理工大學線上教學研究撥款（96 萬港元，2021-22 年）、無人機模擬平台開發的 URIS 本科教學撥款（10 萬港元，2021-23 年），並於 2024 年的[香港理工大學校長傑出教學獎](https://www.polyu.edu.hk/)（2024-26 年）達到高峰。
