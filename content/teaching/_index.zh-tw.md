IPNL 的教學橫跨本科航空工程、以 GNSS 為核心的科目、研究生研究訓練，以及一條歷時多年、針對工程教育的 **以 GitHub 為基礎的教學法** 研究主線。貫穿其中的主旨很簡單：開闊學生的視野，讓他們認識開源與協作的世界，從而駕馭 AI／機器人浪潮 —— 透過培養學會學習、解決問題與協作能力。所教科目自 2017 年起在香港理工大學 e-SFQ 評分中持續位居 **前 10–25 percentile**（平均 eSFQ 約 4.2–4.9），相關工作已在系、院、大學及教資會層級獲得肯定。

## 以 GitHub 為基礎的教學法 —— 標誌性的教學創新

本實驗室標誌性的教學創新研究主線，也是 2024 年校長教學獎案例的基礎。它始於 2020 年、獲 HK$960k 的線上教學發展撥款，經 EDC 與南方科技大學同儕推廣，並於 **ICLT 2025 會議論文** 達到高峰。

**核心理念。** 以 GitHub 為基礎的教學法，是將 **(1) 翻轉課堂** 與 **(2) 專題式學習** 整合，並透過業界實際使用的同一個專業線上程式碼分享與專案管理平台來實踐。當 GitHub 工作流程 —— 版本控制、pull request、議題追蹤、程式碼審查 —— 成為主要的課堂協作環境時，學生所建立的線上協作、解決問題與 **學會學習（L2L）** 能力，遠比傳統的提交平台更為有效。同樣的技能（尋找開源程式碼、與線上協作者開啟專案、共同編寫與審查程式碼）正是雇主與研究日益要求的。

### 量測到的成效（EDC 試點研究）

透過香港理工大學教育發展中心，對一年級 *工程新生研討* 班級進行前測／後測研究加以評估。每一項目標學習屬性都有顯著統計意義的提升：

| 學習屬性 | 之前 | 之後 | 提升 | 顯著性 |
|---|---:|---:|---:|---|
| 線上溝通 | 2.68 | 3.73 | **+39%** | p < 0.001 |
| 線上協作 | 3.02 | 3.79 | **+25%** | p < 0.001 |
| 解決問題 | 3.28 | 4.07 | **+24%** | p < 0.001 |
| 學會學習 | 3.37 | 4.02 | **+19%** | p < 0.001 |

*來源：EDC 試點研究，載於下方的 ICLT 2025 論文。*

### 在課程中的覆蓋範圍

以 GitHub 為基礎的教學法橫跨 **七門本科與研究生科目**：ENG1003 *工程新生研討*、AAE2004 *航空系統與航空運輸法規導論*、AAE3004 *動態系統與控制*、AAE4006 *飛行力學與控制系統*、AAE4002 *畢業專題*、AAE4203 *導引與導航*，以及 AAE6102 *衛星通訊與導航*（RPg）。此方法也延伸至 **研究督導** —— RPg 學生透過 GitHub 製作原型並發布開源（例如 [UrbanNav](https://github.com/IPNL-POLYU/UrbanNavDataset)、[GraphGNSSLib](https://github.com/weisongwen/GraphGNSSLib)、[pyrtklib](https://github.com/IPNL-POLYU/pyrtklib)），同時最大化研究誠信與影響力。

## 案例研究：LifeSparrow —— 從課堂到現場

最清晰的證據，表明此教學法產出的不只是畢業生，更是 **解決真實社會問題的創辦人**。

經由本實驗室專題式教學與 URIS 本科研究管道培育的學生 —— 由 **Max Lee**（現為 AAE 的 HKPFS 博士候選人，自其獲獎的本科實習起便由 Prof. Hsu 指導）領導，並有共同創辦人 **Josua Chan** —— 將他們的 **FJ005** 專題（2021 年 Techathon 智慧城市冠軍）發展為 **[LifeSparrow Solutions](https://www.hkstp.org/en/directory/material-and-precision-engineering/lifesparrow-solutions-limited)**，其 **DIAS** 系統將 AI 人體偵測應用於無人機影像，用於 **山野搜救** —— 將搜索時間縮短 **超過 65%**。

- **真實世界影響：** LifeSparrow 與 **香港消防處** 合作開發 AI 無人機影像搜索系統，用於 **馬鞍山郊野公園** 一宗實際失蹤行山者的搜索（[SCMP](https://www.scmp.com/news/hong-kong/society/article/3237748/hong-kong-aviation-engineering-high-fliers-get-down-earth-help-rescue-teams-find-lost-hikers)）。
- **支持：** 為 **HKSTP 培育公司**，並自香港理工大學、數碼港與香港科學園籌得約 **HK$1.8 million**（[香港理工大學，2024](https://www.polyu.edu.hk/publications/pulse-polyu/issue/202405/research-knowledge-transfer/polyu-alumni-named-forbes-30-under-30-asia-for-revolutionary-search-and-rescue-drone-system)）。
- **肯定：** Max Lee 入選 **富比士亞洲 30 Under 30**（消費科技）2024（[香港理工大學特色校友](https://www.polyu.edu.hk/en/alumni/featured-alumni/alumni-achievements/max-lee-jwo-lem/)）；相關工作獲 **CNN** 報導（[2023](https://www.cnn.com/videos/tv/2023/10/09/ai-first-responders-hnk-spc.cnn) 及一則 2025 年的 ["Tech for Good"](https://edition.cnn.com/2025/10/27/world/video/lifesparrow-ai-drone-hong-kong-tech-for-good-spc-hk) 專題）。

這條脈絡 —— 新生 GitHub 專題 → URIS 研究 → 獲富比士肯定、已落地且能拯救生命的新創公司 —— 是此教學法主旨的具體呈現：*讓學生投入真實問題，他們便不只是學習，而是動手打造。*

> **註：** LifeSparrow 與 Prof. Hsu 目前的其他新創活動不同。在任何對外材料中，請將兩者分開。
