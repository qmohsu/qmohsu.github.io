## 基于 GitHub 的教学法 — 标志性的教学创新

本实验室标志性的教学创新研究主线,也是 2024 年校长教学奖案例的基础。它始于 2020 年那笔 HK$960k 的在线教学发展拨款,在 EDC 与南方科技大学进行了同行推广,并以 **ICLT 2025 会议论文**为高峰。

**核心理念。** 基于 GitHub 的教学法,是将**(1)翻转课堂**与**(2)项目式学习**整合,通过业界实际使用的同一个专业在线代码共享与项目管理平台来实施。当 GitHub 工作流——版本控制、pull request、issue 跟踪、代码评审——成为课堂主要的协作环境时,学生在在线协作、解决问题以及**学会学习(L2L)**等能力上的提升,远比传统提交门户更为有效。这些技能(查找开源代码、与线上协作者发起项目、共同编码与评审)正是雇主和科研日益要求的能力。

### 实测成效(EDC 试点研究)

通过香港理工大学教育发展中心,对一年级*工程新生研讨课*学生群体进行前测/后测研究予以评估。每一项目标学习属性都在强统计显著性下取得提升:

| 学习属性 | 之前 | 之后 | 提升 | 显著性 |
|---|---:|---:|---:|---|
| 在线沟通 | 2.68 | 3.73 | **+39%** | p < 0.001 |
| 在线协作 | 3.02 | 3.79 | **+25%** | p < 0.001 |
| 解决问题 | 3.28 | 4.07 | **+24%** | p < 0.001 |
| 学会学习 | 3.37 | 4.02 | **+19%** | p < 0.001 |

*来源:EDC 试点研究,详见下方 ICLT 2025 论文。*

### 覆盖整个课程体系

基于 GitHub 的教学法横跨**七门本科与研究生科目**:ENG1003 *工程新生研讨课*、AAE2004 *航空系统与航空运输法规导论*、AAE3004 *动态系统与控制*、AAE4006 *飞行力学与控制系统*、AAE4002 *顶石项目*、AAE4203 *制导与导航*,以及 AAE6102 *卫星通信与导航*(RPg)。该方法还延伸至**研究指导**——RPg 学生通过 GitHub 进行原型开发并发布开源成果(例如 [UrbanNav](https://github.com/IPNL-POLYU/UrbanNavDataset)、[GraphGNSSLib](https://github.com/weisongwen/GraphGNSSLib)、[pyrtklib](https://github.com/IPNL-POLYU/pyrtklib)),从而最大化科研诚信与影响力。

## 案例研究:LifeSparrow — 从课堂到现场

这是该教学法不仅培养出毕业生、更培养出**解决真实社会问题的创业者**的最有力证据。

经实验室项目式教学与 URIS 本科生科研管道培养的学生——由 **Max Lee**(现为 AAE 的 HKPFS 博士候选人,自其获奖的本科实习起即由 Prof. Hsu 指导)带领、并与联合创始人 **Josua Chan** 一道——将他们的 **FJ005** 项目(2021 年 Techathon 智慧城市冠军)发展为 **[LifeSparrow Solutions](https://www.hkstp.org/en/directory/material-and-precision-engineering/lifesparrow-solutions-limited)**,其 **DIAS** 系统将 AI 人体检测应用于无人机影像,用于**山岭搜救**——将搜索时间缩短**逾 65%**。

- **现实影响:** LifeSparrow 与**香港消防处**合作开发了一套 AI 无人机影像搜索系统,曾用于**马鞍山郊野公园**一次真实的失踪行山者搜救([SCMP](https://www.scmp.com/news/hong-kong/society/article/3237748/hong-kong-aviation-engineering-high-fliers-get-down-earth-help-rescue-teams-find-lost-hikers))。
- **支持:** 作为 **HKSTP 培育企业**,从理大、数码港及香港科学园筹得约 **HK$1.8 million**([PolyU, 2024](https://www.polyu.edu.hk/publications/pulse-polyu/issue/202405/research-knowledge-transfer/polyu-alumni-named-forbes-30-under-30-asia-for-revolutionary-search-and-rescue-drone-system))。
- **认可:** Max Lee 入选 **福布斯亚洲 30 Under 30**(消费科技类)2024([理大杰出校友](https://www.polyu.edu.hk/en/alumni/featured-alumni/alumni-achievements/max-lee-jwo-lem/));该工作曾获 **CNN** 报道([2023](https://www.cnn.com/videos/tv/2023/10/09/ai-first-responders-hnk-spc.cnn) 及 2025 年一期[“科技向善”](https://edition.cnn.com/2025/10/27/world/video/lifesparrow-ai-drone-hong-kong-tech-for-good-spc-hk)专题)。

这条轨迹——新生 GitHub 项目 → URIS 科研 → 获福布斯认可、已落地部署、帮助挽救生命的初创企业——正是该教学法论点的具体呈现:*让学生投入真实问题,他们便不只是学习,而是去创造。*

> **注:** LifeSparrow 与 Prof. Hsu 当前的其他初创活动有所区别。在任何对外材料中请将两者分开。
